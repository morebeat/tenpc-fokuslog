# Security Policy

## Supported Versions

We currently support the following versions of FokusLog with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We appreciate your effort to responsibly disclose your findings. As a privacy-first application, we take security very seriously.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to:
**kontakt@squirrel-minds.de**

### What to include

*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   Potential impact.

### Our Response

1.  We will acknowledge receipt of your report within 48 hours.
2.  We will investigate the issue and determine its severity.
3.  We will work on a fix and release it as soon as possible.

Thank you for helping us keep FokusLog safe for everyone.
